# CSV to JSON Test Automation

This project automates the testing of the `csvtojson` tool using input space partitioning and combinatorial test generation. It includes scripts to generate test cases, run the `csvtojson` tool, compare the results, and log differences.

## Installation

1. **Install `csvtojson`:**
   - Ensure you have Node.js installed: [Install Node.js](https://nodejs.org/)
   - Install `csvtojson`:
     ```bash
     npm install -g csvtojson
     ```

2. **Run Tests**:
   - Navigate to the project directory and run the provided scripts to generate input files, run tests, and compare outputs.

## Directory Structure

```bash
TestData
├── ActualMessages      # Contains logs comparing actual vs expected messages.
├── ActualOutput        # Stores the actual JSON output generated by `csvtojson`.
├── ExpectedMessages    # Contains logs of expected vs actual output differences.
├── ExpectedOutput      # Stores the expected JSON output (oracle) for each test case.
├── Scripts             # Contains all automation scripts for generating inputs, outputs, and comparisons.
└── TestFiles           # Contains the input CSV files for each test case.
